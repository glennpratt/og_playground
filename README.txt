Sample environment for og and Drupal.

QUICK INSTALL (Drupal 6.x)

wget https://github.com/glennpratt/og_playground/raw/6.x/og6.make
drush make og6.make www

Drush make should build you test environment in the www directory.

QUICK INSTALL (Drupal 7.x)

wget https://github.com/glennpratt/og_playground/raw/7.x/og7.make
drush make og7.make www

Drush make should build you test environment in the www directory.
